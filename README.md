## Sobre o Projeto
Este repositório apresenta o protótipo de baixa fidelidade do aplicativo ReCiclo, desenvolvido na disciplina de Interação Humano Computador e UX.
O objetivo foi representar a estrutura, o fluxo de navegação e a interação entre usuários, utilizando wireframes simples.

## Proposta de Valor
O ReCiclo conecta doadores e interessados em itens reutilizáveis, facilitando o reaproveitamento e reduzindo o desperdício.

## Descrição das Telas

### Tela Inicial

Permite ao usuário escolher entre os perfis de doador ou interessado.

### Tela do Doador
Interface para cadastro de itens com informações básicas como nome, categoria e descrição.

### Tela do Interessado
Exibe os itens disponíveis em formato de lista para navegação.

### Tela de Interação
Representa o contato entre doador e interessado após a demonstração de interesse.

### Tela de Resultado
Indica a conclusão da interação, representando o sucesso da doação.

---

## Jornada do Usuário

1. O usuário acessa o aplicativo
2. Escolhe o perfil (doador ou interessado)
3. O doador cadastra um item ou o interessado navega pelos itens
4. O interessado demonstra interesse
5. O sistema registra a interação
6. A doação é concluída
7. O resultado é exibido
